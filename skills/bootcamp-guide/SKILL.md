# Bootcamp Guide — 训练营引导

> 参考: MeowAI Home bootcamp-guide skill

## 训练营阶段

| Phase | 内容 | 目标 |
|-------|------|------|
| **Phase 1** | 认识猫猫 | 了解三只猫的角色 |
| **Phase 2** | 基础协作 | 学会 #ideate 和 #execute |
| **Phase 3** | 技能使用 | 掌握核心技能触发 |
| **Phase 4** | 完整流程 | 走完一个 Feature 生命周期 |

## 引导原则

1. **循序渐进**: 每次只教一个新概念
2. **做中学**: 边引导边实践
3. **鼓励提问**: 没有傻问题
4. **及时反馈**: 每步都有确认
